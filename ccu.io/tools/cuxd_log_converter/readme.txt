Kleines Tool zum convertieren von CUxD Logs in das CCU.IO Log-Format.

Hinweis: Der Converter löst "Aliase" in CUxD Logs nicht auf.

Anleitung:
* cuxd Logfiles (nur solche mit Zeitstempel im Dateinamen!) im Verzeichnis ccu.io/tools/cuxd_log_converter/logs ablegen
* ins verzeichnis cuxd_log_converter wechseln
* node converter.js aufrufen
* abwarten ;-)

Achtung: Der Converter hängt an vorhandene Dateien an, also aufpassen dass man keine Dopplungen durch mehrfachen Aufruf erzeugt.
